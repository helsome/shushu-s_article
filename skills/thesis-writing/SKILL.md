# 学术论文写作技能

## 概述

本技能用于指导AI大模型应用软件免费增值定价策略研究论文的写作，确保符合学术论文规范和南京财经大学毕业论文格式要求。

## 核心行文原则【重要】

### 一、段落式叙述，禁止滥用分点

学术论文的正文必须是**连贯的段落叙述**，而非分点列表。具体要求：

1. **正文段落化**：每个观点用完整的句子和段落展开，段落之间用过渡句自然衔接
2. **禁止无意义分点**：不得使用"1. 2. 3."或"- - -"罗列观点，除非是列举具体政策措施、对策建议等必须分条的内容
3. **一段一主题**：每个段落围绕一个核心观点展开，段落长度适中（150-300字）

**错误示范**：
```
研究意义包括：
1. 理论意义：丰富定价理论
2. 实践意义：指导企业决策
3. 社会意义：促进行业发展
```

**正确示范**：
```
本研究具有重要的理论意义与实践价值。从理论层面看，生成式AI的高推理成本特性对传统免费增值理论的零边际成本假设构成了根本挑战，本研究通过剖析OpenAI的定价策略，可以丰富和拓展现有的互联网定价理论体系。从实践层面看，OpenAI作为行业领头羊，其定价策略对全球AI企业具有强烈的示范效应，深入研究其策略逻辑可为中国AI企业的商业化路径提供决策参考。
```

### 二、文献综述的段落式写法

文献综述应将多个学者观点融入段落，用"XXX（年份）认为..."串联，而非逐条罗列。

**错误示范**：
```
关于免费增值模式，学者们有如下观点：
- Anderson（2009）认为...
- 王玖河（2022）认为...
- Simon-Kucher（2024）指出...
```

**正确示范**：
```
国内学术界对"免费增值"模式的研究主要从互联网平台经济领域起步，早期侧重于网络效应下的用户转化机制。何忠伟（2005）认为通过实行补贴政策能够减轻用户负担，肖海峰（2006）认为该政策保护了用户利益，韩喜平（2007）则从市场活性角度进行了分析。近年来，随着数字经济与人工智能的深度融合，研究焦点逐渐转向数据要素价值和AI商业模式的特殊性。王玖河等（2022）从价值共创视角指出，用户的信息交互行为是实现价值增值的核心，这为解释AI免费用户的"数据贡献"提供了理论依据。
```

### 三、标题层级规范

严格遵循以下层级，**不得随意增加层级**：

```
一、XXXXXX（一级标题，黑体四号）
（一）XXXXXX（二级标题，正文同字体加粗）
1. XXXXXX（三级标题，仅用于列举政策、措施、对策等）
```

**注意**：三级标题"1. 2. 3."仅在以下情况使用：
- 列举具体政策措施（如"现行主要的粮食补贴政策措施有："）
- 提出对策建议
- 罗列研究方法步骤

### 四、自然过渡与衔接

段落之间必须有过渡句，常用过渡方式：

- **背景引入**："在此背景下"、"随着...的发展"
- **递进深入**："进一步分析"、"更重要的是"、"此外"
- **转折对比**："然而"、"但是"、"与此不同的是"
- **因果总结**："由此可见"、"综上所述"、"可见"
- **数据引出**："由图X可以看出"、"由表X得出"、"根据数据表明"

### 五、数据与图表的融入

图表应融入正文，前有引导句，后有分析段：

```
邹区镇粮食补贴政策实施情况如表1所示。

表1 邹区镇2013年-2016年粮食直接补贴情况表
[表格内容]
资料来源：邹区镇财政局内部资料

由表1得出，从2013年至2016年，邹区镇水稻种植粮食直补补贴户数逐年上升，确认的应补贴面积也呈上升趋势。特别是2016年补贴面积较2015年上升了21.68%，可见粮食直补政策普及度在扩大。
```

## 论文结构规范

### 整体结构

封面 → 原创声明 → 目录 → 摘要（中英文）→ 正文（六章）→ 参考文献 → 附录

### 各章内容要求

**一、引言**：研究背景、目的意义、研究方法、论文结构（连续段落，不分小节）

**二、国内外研究综述**：
- （一）国内相关研究综述（段落式综述多位学者观点）
- （二）国外相关研究综述（段落式综述多位学者观点）

**三、研究对象现状分析**：
- （一）研究对象概况
- （二）发展演进
- （三）现行政策/策略/措施

**四、实施效果/案例分析**：
- 按维度分小节，每节段落式论述，配合图表数据

**五、存在的问题**：
- 每个问题一个小节标题，内容段落式展开

**六、对策建议**：
- 可使用"（一）（二）（三）"分条，每条内容仍需段落式论述

## 格式规范

### 正文格式
- 中文正文：仿宋，小四
- 英文/数字：Times New Roman，小四
- 行距：28磅
- 页边距：上2.5cm，下2.5cm，左3.0cm，右2.0cm

### 标题格式
- 一级标题：黑体四号
- 正文：仿宋小四

### 摘要格式
- 中文摘要：200-300字，仿宋五号
- 关键词：3-4个，分号分隔
- 英文摘要：Times New Roman五号

### 参考文献格式（GB/T 7714）
- 期刊：[序号] 作者.题目[J].刊名,年,(期):页码.
- 专著：[序号] 作者.书名[M].出版地:出版社,年.
- 中文在前，英文在后，各自按第一作者姓氏排列

### 图表规范
- 表名在表上方，图名在图下方
- 必须标注"资料来源"
- 编号连续（表1、表2...；图1、图2...）

## 写作检查清单

### 行文检查
- [ ] 正文是否为段落式叙述（非分点列表）
- [ ] 文献综述是否将多位学者观点融入段落
- [ ] 段落之间是否有过渡句衔接
- [ ] 标题层级是否符合规范（一、（一）、1.）
- [ ] 图表前后是否有引导句和分析段

### 格式检查
- [ ] 字体字号是否正确
- [ ] 参考文献格式是否规范
- [ ] 图表是否标注来源
- [ ] 页码是否连续
